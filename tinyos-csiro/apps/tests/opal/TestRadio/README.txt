README for TestRadio
Author/Contact: philipp.sommer@csiro.au

Description:

TestRadio sends active message broadcasts at 1Hz and briefly flickers LED 0 
whenever it has sucessfully sent a broadcast. Whenever it receives one of 
these broadcasts from another node, it toggles LED 1. Furthermore, it prints
the corresponding source address, sequence number and RSSI/LQI values of the
packet to the serial port.

Tools:

None.

Known bugs/limitations:

None.
